# JavaScript - 布尔
**Boolean** 对象表示两个值，不是“真”就是“假”。

## 语法

创建一个 **Boolean** 对象：

  ```
     var val = new Boolean(value);
  ``` 
 
如果省略 *value* 参数或参数是 0，-0 ，空，假， NaN ，未定义，或者是空字符串（“”），这个对象初始值为假。
